css 특이성(특수성)

css+동일한 요소를 여러 개의 css 규칙이 적용될 때 
우선순위 결정시 사용되는 특성: 특이성
특이성은 점수로 정해진다

1.인라인식 스타일 시트 적용방법 1000점
2. *              0점
	태그명, 의사요소 : 1점
	클래스명, 의사클래스명, 속성선택자:10점
	#아이디명:              100점
	
	예)
	#navbar p#demo         : 100+1+100
	{
	color:red;
	}
	#demo                  :100
	{
	color:blue
	}
	
	
	예외
	!important 규칙
	행당 요소의 특정 속성에 대한 모든 이전 스타일 재정의한다
